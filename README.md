# Todo App - Frontend

This is the **frontend** for a Todo application. The app connects to a backend hosted on Render for managing tasks.

## Features

- Add, edit, and delete tasks.
- Mark tasks as completed.
- Responsive UI.

## Tech Stack

- **Frontend**: React 
- **Backend**: Hosted on [Render](https://render.com)
